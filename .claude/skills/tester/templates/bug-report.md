```

Severity: Blocker / Critical / Major / Minor / Trivial

Environment: Dev / Staging / Production
Browser/Device: [Details]

Steps to Reproduce:
1. [Step]
2. [Step]
3. [Step]

Expected Result: [What should happen]
Actual Result: [What actually happens]

Screenshots/Logs: [Attached]
Workaround: [If any]

Root Cause Analysis: [If known]
Regression Risk: [Assessment]
Affected Users: [Scope of impact]
```

## Severity Guide

| Severity | Definition | Example |
|----------|-----------|---------|
| **Blocker** | App unusable, data loss | App crashes on install |
| **Critical** | Core feature broken, no workaround | Billing not working |
| **Major** | Core feature broken, workaround exists | Export fails but manual copy works |
| **Minor** | Non-core feature issue | Sorting not working on optional list |
| **Trivial** | Cosmetic, typo | Button text misaligned |
